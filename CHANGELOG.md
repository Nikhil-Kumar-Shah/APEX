## [1.0.0] - 2026-07-20

### Added
- **Core Configuration**: Schema validation, workspace loaders, Google Drive integrations, and startup wizards.
- **Model Engine**: Adapters for Transformers, vLLM, and llama.cpp, cache loaders, and HF downloaders.
- **Server API**: FastAPI implementation with OpenAI-compatible endpoint routing and SSE streaming completions.
- **Workspace Memory**: Workspace isolation setups, conversation JSON logger, and AST repository codebase indexer.
- **User Interface**: Notebook tab dashboard, server buttons, log search lists, resource monitoring, and benchmarks metrics.
- **Bootstrap Loader**: GitHub-first bootstrap script to fetch repositories, resolve pip requirements, and switch tags.

# Security Policy

## Supported Versions

We actively update and provide security patches for the following versions:

| Version | Supported |
| :--- | :--- |
| 1.0.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

We take the security of **APEX** seriously. If you identify a security vulnerability (such as credential leaks, directory traversals, or command injections), please do **NOT** open a public issue.

Instead, please contact me directly via the contact form at https://www.nikhilkshah.me/contact.

Please include:
* Detailed steps to reproduce the vulnerability.
* Environmental metrics (OS version, Hugging Face Hub library version).
* Potential impact scope.

We will review your submission and respond within 48 hours to coordinate a security fix.

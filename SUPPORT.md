## Community Q&A and Discussions

For questions, configuration help, how-to guides, and general discussion:

👉 **[GitHub Discussions](https://github.com/Nikhil-Kumar-Shah/APEX/discussions)**

Before opening a new discussion:
- Search existing discussions — your question may already be answered.
- Use descriptive titles so others can find the thread.
- Share your `apex.config.json` (with sensitive values removed) when asking about configuration issues.

## Bug Reports

If you have found a reproducible bug in APEX:

👉 **[Open a Bug Report](https://github.com/Nikhil-Kumar-Shah/APEX/issues/new?template=bug_report.md)**

A good bug report includes:
- The exact steps to reproduce the issue.
- What you expected to happen vs. what actually happened.
- Relevant logs from the notebook console output.
- Your environment: Colab vs. local, Python version, GPU type, APEX version.

## Security Vulnerabilities

If you have found a security vulnerability, **do not open a public issue**.

Please report it privately using the contact form at:
👉 **[https://www.nikhilkshah.me/contact](https://www.nikhilkshah.me/contact)**

See [SECURITY.md](SECURITY.md) for the full disclosure process.
